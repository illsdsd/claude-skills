- "远程服务器"、"服务器"、"服务器上"、"ECS"

## 执行方式

### 基本连接命令

```bash
ssh -o ConnectTimeout=10 root@8.140.221.24 "<命令>"
```

### 文件传输

```bash
# 上传文件到服务器
scp <本地文件> root@8.140.221.24:<远程路径>

# 下载文件到本地
scp root@8.140.221.24:<远程路径> <本地文件>
```

### 执行本地脚本到远程

```bash
ssh root@8.140.221.24 'bash -s' < <本地脚本>
```

## 注意事项

1. 服务器 root 密码已在聊天中暴露过，强烈建议用户修改密码
2. 所有远程命令通过本地 SSH 发起，不会在服务器上保存任何凭证
3. 执行文件编辑时，先在本地写好内容，再通过 `scp` 或 `ssh 'cat > file'` 推送到服务器
4. CentOS 7 已 EOL，安装软件优先使用 EPEL 仓库
